# Smooth Sneaking Mod

Bring back smooth sneaking from the future and the past.

A Minecraft 1.8.9 mod that implements smooth sneaking animation.
This only adds visual effect and will not change your behavior at server, so it's not a cheat.

## Dependency

- [UniMixin](https://github.com/LegacyModdingMC/UniMixins)

Although this project depends on SpongeMixin only, we recommend you to use UniMixin for less complexity.

UniMixin 1.7.10
